# 🚀 QUICK START GUIDE - Deploy to GitHub.io in 5 Steps

## Your Wedding Invitation Website is Ready! 🎉

All files are created and ready to deploy. Follow these simple steps:

---

## STEP 1: Create GitHub Account (if you don't have one)
- Go to [github.com](https://github.com)
- Sign up for free
- Verify your email

---

## STEP 2: Create a New Repository
1. Click the **+** icon (top right) → **New repository**
2. **Name it EXACTLY:** `yourusername.github.io`
   - Example: `sumit2026.github.io` (replace with YOUR username)
3. **Description:** Wedding Invitation (optional)
4. ✅ **Make it PUBLIC**
5. Click **Create repository**

---

## STEP 3: Upload Your Files to GitHub

### Option A: Using Git Command (Recommended)

If you have Git installed:

```bash
# Open Command Prompt/PowerShell and navigate to wedding-invitation folder
cd d:\pyproj\wedding-invitation

# Initialize git
git init

# Add all files
git add .

# Create commit
git commit -m "Wedding invitation website for Sumit & Sonam"

# Add remote (replace YOUR_USERNAME with your GitHub username)
git remote add origin https://github.com/YOUR_USERNAME/YOUR_USERNAME.github.io.git

# Push to GitHub
git branch -M main
git push -u origin main
```

### Option B: Upload via GitHub Website (Easiest)

1. Go to your new repository: `github.com/YOUR_USERNAME/YOUR_USERNAME.github.io`
2. Click **Add file** → **Upload files**
3. Drag and drop these files:
   - index.html
   - style.css
   - script.js
4. Click **Commit changes**

---

## STEP 4: Add Wedding Music (IMPORTANT!)

1. Download royalty-free wedding instrumental music (MP3):
   - [Pixabay Music](https://pixabay.com/music/) - FREE
   - Search: "Wedding Instrumental" or "Romantic Piano"
   - Download as MP3

2. Name it exactly: `wedding-music.mp3`

3. Upload to your GitHub repository:
   - Click **Add file** → **Upload files**
   - Upload `wedding-music.mp3`
   - Click **Commit changes**

---

## STEP 5: View Your Website

✅ **Your website is now LIVE!**

Visit: `https://YOUR_USERNAME.github.io`

Example: If your GitHub username is `sumit2026`, your site is at:
`https://sumit2026.github.io`

## 🎵 Music Won't Play?

**Common Solutions:**

1. ✅ Make sure `wedding-music.mp3` is uploaded to the repository
2. ✅ Wait 5 minutes for GitHub to sync
3. ✅ Refresh the page (Ctrl+F5 or Cmd+Shift+R)
4. ✅ Click the speaker icon (🔊) in top-right to enable music
5. ✅ Check if your browser allows audio autoplay
