## 🚀 How to Deploy to GitHub Pages

### Quick 5-Step Process:

1. **Create GitHub Account** - Go to github.com and sign up (free)

2. **Create New Repository** - Name it `yourusername.github.io`
   - Replace `yourusername` with your actual GitHub username
   - Make it PUBLIC

3. **Upload Files** - Two options:
   - **Option A (Recommended):** Use Git commands from command line
   - **Option B (Easiest):** Drag & drop files on GitHub website

4. **Add Wedding Music** - Download free music and upload as `wedding-music.mp3`
   - Download from Pixabay or YouTube Audio Library
   - Must be named `wedding-music.mp3`

5. **View Your Website** - Your site is now live at:
   - `https://yourusername.github.io`

## 🆘 Troubleshooting

**Music not playing?**
- Ensure `wedding-music.mp3` is uploaded to the repository
- Wait 5 minutes for GitHub to sync
- Click the speaker icon to enable music
- Refresh page (Ctrl+F5)

**Website not showing?**
- Check repository is named `yourusername.github.io` (exact match)
- Ensure repository is set to PUBLIC
- Wait 1-2 minutes for GitHub Pages to deploy
- Clear browser cache

**Scratch effect not working?**
- Works best on desktop with mouse
- On mobile, use finger to scratch
- Make sure JavaScript is enabled in browser
